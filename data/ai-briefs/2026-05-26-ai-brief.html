Most Important Items

Developer tooling

Datasette turns navigation into an extensibility surface

Fact: Datasette 1.0a30 adds a visible "Jump to..." menu, expands search beyond tables, renames the backing endpoint to /-/jump, adds jump_items_sql(), and adds makeJumpSections().

Analysis: This matters because it turns plugin-discovered data and agent actions into first-class navigation. For data apps, "where do I go next?" is becoming a programmable interface.

Confidence: high. Verified via Simon Willison post, Datasette blog, docs, and GitHub release.

Agent UX

datasette-agent moves into the command palette

Fact: datasette-agent 0.1a4 now integrates with Datasette 1.0a30's Jump menu; Simon's post says signed-in users can start a new agent chat from that menu at agent.datasette.io.

Inference: Agent entry points are likely to migrate from chat-only surfaces into host-app control planes: search boxes, command palettes, debug menus, and admin consoles.

Confidence: high for the release fact, medium for broader UX trend.

X + GitHub

OpenClaw posts emphasize dependency and prompt-budget discipline

Fact: Peter Steinberger posted that OpenClaw replaced Sharp/Jimp with Photon, a Rust/WebAssembly image-processing library; Photon describes itself as cross-platform Rust/WASM image processing on GitHub. He also linked a skill-cleaner skill that audits skill prompt budget, duplicates, unused skills, and long descriptions.

Analysis: The pattern is concrete: agentic desktop tools are being optimized on two axes at once, binary/runtime footprint and context footprint. Both are product quality issues, not just engineering hygiene.

Confidence: medium-high. GitHub sources verified; X context limited to public embed text and expanded links.

Model roadmap

Fact from public X embed: Elon Musk said Grok foundation model V9-Medium had finished training, claimed a 1.5T scale, cited Cursor data in supplementary training, and projected public release in 2-3 weeks after fine-tuning and RL.

Uncertainty: No model card, eval report, pricing, safety details, or release artifact was reachable. Treat scale, eval quality, and release timing as issuer claims only.
